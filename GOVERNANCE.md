# ACES Project Governance

The ACES project aims to be governed in a transparent, accessible way for the benefit of the community. Participation in this project is open to all. 

As a project of the Academy Software Foundation, ACES builds off ASWF governance policies. The ASWF foundation documents house the the [**ACES project charter**](https://github.com/AcademySoftwareFoundation/foundation/blob/main/project_charters/aces_charter.pdf).

## Project roles

There are three primary project roles:
- [Contributor](#contributor) - Submits ideas and/or code
- [Committer](#committer) - Reviews contributions
- [Technical Steering Committee](#technical-steering-committee) - Directs project strategy

## Contributor

The Contributor role is the starting point for anyone participating in the project. Typical activities of a Contributor might include:

- Participating in working groups
- Contributing code and/or documentation changes that improve the project
- Identifying and logging issues
- Submitting ideas or feature requests

## Committer

A Committer is a Contributor who has earned the authority to commit code directly to the respository and approve submissions. Typical activities of a Committer are the same as a Contributor, but also include:

- Reviewing and commenting on issues and merging pull requests
- Assisting users via our communication platforms
- Guiding novice contributors to be successful

To become a Committer, a Contributor must demonstrate an understanding of the codebase through contributions and engagement on the community channels. Candidates are nominated by existing Committers based on a track record of high-quality contributions and then approved by a simple majority vote of the TSC.

## Technical Steering Committee

The Technical Steering Committee (TSC) is the leadership group of ACES. Its primary responsibilities include:

- Determining the overall strategic direction of the project
- Ensuring the project community has the necessary resources and infrastructure
- Providing updates to the ASWF TAC and the broader ACES community

### TSC Meetings
All TSC meetings are open to the public. Meetings are scheduled every other Tuesday at 10:00 AM Pacific Time via Zoom. You can find the upcoming schedule and Zoom link on the [ASWF's ACES calendar](https://zoom-lfx.platform.linuxfoundation.org/meetings/aces).

See the [TSC meeting notes archive](https://lf-aswf.atlassian.net/wiki/external/ZmE2YzVjZGYzYWFhNDRmZGJkNmE4MWZkODRmN2JkZDE) to review discussion in previous meetings.
